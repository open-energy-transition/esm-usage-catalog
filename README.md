# ⚡ Energy System Model Use-Case Catalogue

This repository documents how ministries, regulators, grid operators, and research institutions use energy system modelling frameworks in real-world decision-making.

---

## 🎯 Purpose

This application provides a structured overview of real-world applications of energy system models. It is designed to support:

- Researchers exploring modelling practices
- Funders assessing impact and adoption
- Practitioners identifying relevant tools and use cases

---

## 🌍 Features

Key features of the application include:

- Geographic analysis of modelling usage
- Tool and category breakdown
- Institutional insights
- Evidence strength classification
- Interactive filtering and exploration
- Downloadable dataset

---

## 📊 Data

The dataset consists of curated records of real-world energy modelling use cases. Each entry captures:

- Country or region
- Institution
- Modelling tool
- Use case and application context
- Evidence source and strength
